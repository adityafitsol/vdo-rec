Two ways to record.

Choose between the raw power of the terminal or the visual precision of our browser dashboard.

terminal

Terminal-Based

Lightning-fast commands, flag-based control, and a streamlined interactive wizard.
web

GUI-Based

Modern visual dashboard with live meters, timers, and a gallery of your recordings.

1. Terminal Mastery

Interactive Mode (Recommended)

Simply type the command with no arguments to launch the setup wizard. It will guide you through selecting your monitor, microphone, and system audio.

$ vdo

Direct Flag Control

Bypass the wizard by using flags for immediate recording.

Common Commands
vdo --screen Record screen only
vdo --screen --mic Add microphone
vdo --screen --mic --sys-audio Record EVERYTHING
vdo --screen --fps 60 High motion

2. Visual Control

Launching the Dashboard

Start the local web server to manage your recordings from any browser. This is the easiest way to monitor levels and manage your library.

$ vdo --gui
equalizer Live Monitoring

Watch real-time volume meters for your microphone and system audio to ensure perfect levels before you capture.

photo_library Recording Gallery

Instant access to all your local `.mp4`, `.webm`, and `.gif` files with one-click playback and management.
